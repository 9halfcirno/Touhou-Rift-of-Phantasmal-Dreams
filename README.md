## 简介 - Description
东方幻梦裂隙（Touhou Rift of Phantasmal Dreams）是一个以**东方Project**为主题的同人游戏

## 计划 - Plan
> **注意⚠：** 请不要对该项目有过高的期待, 也不要认为该项目最终能够落地
> 
> 开发者仅有我自己一人, 该项目更多是积累经验

目前进度: **引擎开发阶段**

<span title="我是个学生没太多时间开发qwq">第一个可玩Demo时间未知</span>

预计玩法将会是2.5D冒险+Rouguelike+弹幕射击游戏+RPG(?)

玩家可以操控不同的角色, 去被梦境裂隙影响的幻想乡, 调查并解决这场异变!

## 技术栈
- 引擎架构: <span title="实际上是参考了Minecraft BE">ECS + 混合传统架构</span>
- 编程语言: <span title="原本是Javascript">Typescript</span>
- 主依赖库: [PixiJS](https://pixijs.com/), [Three.js](https://threejs.org/), [Neutralinojs](https://neutralino.js.org/)
- 构建工具: [Vite](https://vitejs.dev/)
